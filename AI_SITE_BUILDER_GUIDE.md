# AI Site Builder

An admin tool to generate an editable page layout from a short prompt, using the same component system as the Visual Page Builder.

## Where to find it

- Admin: `/admin/ai-site-builder`

## How it works

1. Enter a brief prompt (purpose, audience, offer, tone, location).
2. Click “Generate Page”.
3. The tool calls `/api/site/generate` (Gemini) to produce a component array.
4. The result loads into the Visual Editor so you can tweak layout and content.

## API

- Endpoint: `POST /api/site/generate`
- Body:
  - `prompt: string` (required)
  - `style?: string` (optional tone/style hint)
  - `wordCount?: number` (optional target word count)
- Response:
  - `title: string`
  - `suggestedSlug: string`
  - `notes?: string`
  - `components: PageComponent[]` (compatible with `components/VisualEditor.tsx`)

## Notes

- Only a safe subset of components is generated by default (hero, text, services, testimonials, FAQ, pricing, map, CTA, SEO footer).
- The route enforces AI-enabled settings and handles leaked API key errors with a 403.
- You can continue editing normally in the Visual Editor and export or integrate into your existing pages.
